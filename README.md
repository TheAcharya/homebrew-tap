# Homebrew Tap

This repository contains the homebrew formulas for our applications.

## Usage

### With [Homebrew](https://brew.sh/)

```bash
$ brew tap TheAcharya/homebrew-tap
```

```bash
$ brew uptap TheAcharya/homebrew-tap
```

```bash
$ brew install --cask <formula>
```
